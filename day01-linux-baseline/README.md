# Day 01 - Linux Security Baseline

Establish a security baseline before applying any hardening controls

## Skills Learned

- System enumeration
- Service discovery
- Port auditing
- SSH assessment
- Permission auditing

## Challenge

Create a baseline report and identify:

- Open ports
- Running services
- UID 0 accounts
- SSH weaknesses
- World writable files
- SUID binaries
